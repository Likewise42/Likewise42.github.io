About

Hello! I'm Ryan Muskopf and I'm a recent Game Design and Development graduate at the Rochester Institute of Technology in New York. I am a web developer, that specializes in Javascript with Node.js and React.

Projects

The Caller Queue

The Caller Queue is a system for live shows to stage viewers before going on air. I worked on the third (and most recent) version which utilized Node.js, React, Redux, Socket.io, WebRTC, and MySQL. My work on the Caller Queue has been used by shows such as MLB's Chatting Cage and Daily Blast Live.

Orbit Rings

A game about playing bumper cars using planets! Used Socket.io for real time communication. Worked with Ryan Spencer and Muriel Hagge.

Mega Moonlings

Mega Moonlings is a cookie-clicker inspired application that is an exercise in storing and retrieving data from a Mongo database using Mongoose as well as using React and Bootstrap to create a pleasant user experience.

Pong

My own take on Pong featuring real time networked multiplayer powered by Socket.io!

Lucio Audio Visualizer

An Audio Visualizer inspired by Lucio from Overwatch! Made in Canvas using ES5.

Not Pokemon Go

An excercise in using multiple APIs concurrently. Made by combining the PokeAPI and Google Maps API to create an experience similar to Pokemon GO. Works best on mobile devices.

Port Town

A web game made entirely in Canvas and ES5. This symmetrical game focuses on taking control of the city by placing and attacking your lackys around the map. Not mobile compatible.

NR Debt Relief

A volunteer project done for the North Rockland Debt Relief organization. Fully HTML and CSS with embedded Google Forms to allow for the quick recruitment of volunteers.
